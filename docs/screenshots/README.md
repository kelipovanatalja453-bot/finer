# Finer OS 截图指南

本文档说明需要截取的界面截图，用于 GitHub README 和项目文档。

## 截图列表

### 1. Dashboard 主界面
**文件名**: `dashboard-main.png`
**位置**: 首页 L2 知识库视图
**内容**: 显示文件卡片网格、侧边栏导航、检查器面板
**尺寸**: 1920x1080 或更高

### 2. 六层流水线视图
**文件名**: `pipeline-flow.png`
**位置**: 侧边栏切换不同层级
**内容**: 展示 L0→L1→L2→L3→L5→L6 的完整流程
**尺寸**: 1920x1080

### 3. 数据源配置界面
**文件名**: `data-sources.png`
**位置**: Integrations Hub 或 Data Sources 菜单
**内容**: 微信公众号扫码登录、B站视频转录配置
**尺寸**: 1200x800

### 4. RLHF 评价面板
**文件名**: `rlhf-panel.png`
**位置**: L6 复核台点击文件打开
**内容**: 中文评价界面，显示原文、标的评价、方向评价、评分
**尺寸**: 1200x900

### 5. KOL 评价卡片
**文件名**: `kol-rating-card.png`
**位置**: /demo/kol-rating 页面
**内容**: 星级评级、维度仪表盘、业绩时间线、专注领域
**尺寸**: 1200x1000

### 6. 观点时间线
**文件名**: `opinion-timeline.png`
**位置**: 观点追踪页面
**内容**: 横向时间线，节点显示方向和验证状态
**尺寸**: 1600x600

### 7. 检查器面板
**文件名**: `inspector-panel.png`
**位置**: 右侧面板
**内容**: 文件详情、元数据、操作按钮
**尺寸**: 400x800

### 8. Trade Action 提取结果
**文件名**: `trade-action-extraction.png`
**位置**: L5 抽取台
**内容**: 显示提取的结构化 TradeAction JSON
**尺寸**: 1000x700

## 截图工具推荐

- macOS: `Cmd + Shift + 4` 区域截图，`Cmd + Shift + 5` 窗口截图
- 浏览器开发者工具: 模拟不同设备尺寸
- 截图后可使用 Preview 或 Figma 添加标注

## 截图要求

1. **清晰度**: 使用高分辨率，避免模糊
2. **隐私**: 隐藏敏感信息（API Key、真实数据）
3. **一致性**: 统一使用深色/浅色主题
4. **标注**: 必要时添加箭头或文字说明
