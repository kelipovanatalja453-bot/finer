# 快速上手 | Quick Start Tutorial

本教程将引导你完成从环境配置到运行第一个 Dry-run 任务的全过程。
This tutorial will guide you through the process from environment setup to running your first dry-run task.

## 1. 安装 | Installation
参考 [README](../../README.md) 进行基础安装。
Refer to [README](../../README.md) for basic installation.

## 2. 准备数据 | Prepare Data
将你的图片或音频放入 `data/raw` 目录。
Place your images or audio files into the `data/raw` directory.

## 3. 执行 Dry-run | Run Dry-run
运行以下命令验证流水线：
Run the following command to verify the pipeline:
```bash
finer dry-run --source-dir "./data/raw/trader_ji"
```
