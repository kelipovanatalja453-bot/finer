# AGENTS.md - World Cup ML Projects Research

## Scope

This directory stores external research notes about FIFA World Cup football analytics, machine learning, deep learning, and simulation projects.

It is documentation only. It does not define or modify the Finer OS F0-F8 production pipeline.

## Directory Structure

- `README.md`: index, ranking, and reading path.
- `kaggle-*.md`: Kaggle dataset or notebook analysis.
- `github-*.md`: GitHub repository analysis.

## File Template

Each project note should cover:

1. Source and project identity.
2. Problem definition.
3. Data inputs and feature design.
4. Modeling principle.
5. Implementation workflow.
6. How to use or reproduce.
7. Quality assessment and risks.
8. What Finer can learn from it.

## Boundaries

- Do not copy large third-party source code into this directory.
- Link to original Kaggle or GitHub pages instead.
- Mark predictions as educational or experimental unless the source provides rigorous out-of-sample evidence.
- Avoid introducing deprecated Finer architecture naming.
